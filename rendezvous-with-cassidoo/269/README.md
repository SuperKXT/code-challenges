## Description

Write a function that truncates words in a string to length `n`.

## Example

```ts
const n = 3

$ truncateWords('never gonna give you up', n)
$ 'nev gon giv you up'

$ truncateWords('*hello* darkness, my ~old_friend', n)
$ '*hel* dar, my ~old_fri'
```
