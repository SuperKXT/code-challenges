# Find sum of odd square numbers less than n

[issue #299 of rendezvous with cassidoo.](https://buttondown.email/cassidoo/archive/6053/)

## Description

**Sum the odd-square numbers less than a given integer n.**

Example:

```ts
> oddSquareSum(1)
> 0

> oddSquareSum(2)
> 1

> oddSquareSum(9)
> 1

> oddSquareSum(10)
> 10

> oddSquareSum(44)
> 35
```
